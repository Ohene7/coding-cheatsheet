Tables Cheatsheet

<Tables> can be intimidating. Remember how to create them by studying these notes.

Tags Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The bolded headers for all columns.
<tbody> Table Body Signifies the tabular data of the table. Semantic.
<tfoot> Table Footer Signifies the end of the table, including total sums. Semantic.
<tr> Table Row Add rows to your table with this. Use with th and td.
<th> Table Header Add headings to your table with this. Use with tr.
<td> Table Data The individual cells nested within tr.
